# Worthy — Your worth, made visible.

A living portfolio platform for fresh graduates where every post builds
an evidence-backed profile — automatically, through AI extraction.

## Live Demo
https://worthy-nu.vercel.app/

## How to run locally
copy the link above and paste in any browser will do

## Key screens
- Sign in → select Candidate or Employer role
- Onboarding → 3-part AI questionnaire
- Feed → post an experience, watch AI extract your skills live
- My Portfolio → quick-scan header, skills by category, evidence quotes
- Ask Worthy → positioning, pathways, mentor matching
- Employer toggle → scout candidates, filter by skill, view full profiles

## Built with
- Vanilla HTML / CSS / JavaScript (single file, no framework)
- Claude API (claude-sonnet-4-6) for all AI features
- In-memory state for prototype; Supabase planned for production
